# Global-Super-Store-Analysis.
# Análisis de Inteligencia Comercial: Global Super Store 📊

![Portada Global Super Store](img/global%20super%20store.png)

Este proyecto consiste en un análisis integral de un dataset global de ventas multicanal utilizando **Google Sheets**. El objetivo principal fue transformar datos crudos en un sistema de toma de decisiones estratégico, identificando fugas de beneficios y oportunidades de expansión.

## 🚀 Resumen del Proyecto
Se realizó una auditoría profunda que permitió visualizar un crecimiento total de **+$39.8M** entre 2011 y 2014, con un promedio de ventas anuales de **$61,059,040.75**.

## 🛠️ Herramientas y Metodología
*   **Limpieza de Datos (ETL):** Eliminación de duplicados, corrección de formatos y verificación de campos clave.
*   **Funciones Avanzadas:** Uso de `=UNIQUE()` para categorías, `=SUMIFS()` para segmentación condicional y cálculos de crecimiento porcentual.
*   **Business Intelligence:** Creación de un **Dashboard Interactivo en Google Sheets** con diseño en formato "Z", filtros dinámicos y navegación mediante botones.

## 📈 Insights Clave
*   **Dominio de Categoría:** *Office Supplies* representa más del **60% de las ventas**, siendo el segmento más rentable.
*   **Logística:** El método *Standard Class* gestiona el 60% del volumen total de envíos.
*   **Geografía:** La región *Central* lidera en ventas, mientras que *Caribbean* presenta los márgenes más altos en proporción al costo.

## 🎯 Objetivos SMART Propuestos
Basado en el análisis, se definieron los siguientes KPIs estratégicos:
1.  **Ventas:** Aumentar en un **12%** las ventas en subcategorías líderes (Binders, Storage, Art).
2.  **Logística:** Optimizar entregas urbanas para incrementar en un **10%** el uso de *Same Day*.
3.  **Expansión:** Incrementar en un **20%** la presencia en mercados con potencial (África y EMEA).
